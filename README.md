# parking-er-diagram
ER Diagram for Comic-Con Parking System

This ER diagram represents the database design for a Comic-Con Parking System.

Key Features:

- Vehicle and Vehicle Type management
- Parking Session tracking (entry & exit time)
- Parking Spot allocation with Zone and Level
- Ticket generation for each session
- Payment tracking linked to sessions
  
## Relationships Overview
- One vehicle can have multiple parking sessions
- Each session is linked to one parking spot
- Each session generates one ticket
- Each session has one payment record
